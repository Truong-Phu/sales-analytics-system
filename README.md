# Hệ thống thu thập, quản lý và phân tích dữ liệu bán hàng đa kênh

## Mục tiêu
Xây dựng hệ thống hỗ trợ thu thập, lưu trữ, quản lý và phân tích dữ liệu bán hàng từ nhiều kênh.

## Công nghệ
- Backend: ASP.NET Core
- Database: PostgreSQL
- Analytics: Python (Pandas)
- BI: Power BI

## Chức năng chính
- Quản lý người dùng & phân quyền
- Tích hợp dữ liệu đa kênh
- Thống kê & dashboard
- Báo cáo bán hàng

## Hướng phát triển
- Phân tích nâng cao
- Ứng dụng AI hỗ trợ quyết định

## Run Backend
- cd backend/SalesAnalytics.API
- dotnet run
- API: https://localhost:7119/swagger
  
## Run Frontend
- cd frontend
- npm install
- npm start
- Frontend: http://localhost:3000
